# How Shazam Identifies Songs in 5 Seconds: Audio Fingerprinting with Python

This talk explains how audio fingerprinting works by walking through the ideas behind Shazam-style song recognition and implementing the core concepts in Python.

The goal is to make the "how does Shazam know this song so fast?" question feel concrete: turn audio into a spectrogram, extract stable peaks, create fingerprints, and match a short recording against a known song database.
