# Building Harry Potter's Mirror of Erised with Python and Generative AI

This talk explores a playful generative AI project inspired by the Mirror of Erised, showing how Python can be used to combine user input, image generation, and application logic into an interactive experience.

## Presented At

- PyCon Sweden 2025, Stockholm
